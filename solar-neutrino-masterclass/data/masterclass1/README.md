# Masterclass 1 Data

These files are the public inputs for the first solar-neutrino masterclass.

- `solar_fluxes.csv`: total solar-neutrino fluxes at Earth.
- `energy_spectra.csv`: normalized source spectra and line shapes.
- `nu_electron_recoil_cross_sections.csv`: differential recoil cross sections for neutrino-electron scattering.

The expected SK-like event spectrum is not included. It is the result to be computed during the masterclass.
