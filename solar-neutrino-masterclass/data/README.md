# Masterclass Data

Analytic formulas in the teaching material use natural units, $\hbar=c=1$. CSV files keep practical units in their column names and descriptions.

The main student dataset is in:

```text
data/student/
```

It is generated with:

```bash
python scripts/generate_student_datasets.py
```

These tables have consistent column names. Real tables generated by the teacher with PEANUTS or a chosen solar model can replace the same files if they keep the same columns.

Teacher-side PEANUTS outputs are in:

```text
data/teacher/
```

They are generated by `scripts/generate_peanuts_solar_spectrum.py` and documented
in `data/teacher/README.md`.

## Main Files

- `student/solar_fluxes.csv`: total source fluxes at Earth.
- `student/radial_production_profiles.csv`: production density and shell contribution.
- `student/energy_spectra.csv`: normalized energy spectra.
- `student/peanuts_solar_energy_spectrum.csv`: PEANUTS continuum spectra multiplied by source flux.
- `student/peanuts_solar_line_sources.csv`: PEANUTS line-source markers for the lecture spectrum plot.
- `student/solar_electron_density.csv`: electron-density profile.
- `student/oscillation_probabilities.csv`: $P_{ee}$ in vacuum, daytime, and nighttime.
- `student/earth_regeneration.csv`: Earth correction by nadir-angle bin.
- `student/nu_electron_recoil_cross_sections.csv`: differential cross sections in electron recoil energy.
- `student/mass_state_cross_sections.csv`: total cross sections for mass states.
- `student/detector_response.csv`: SK-like efficiency table.
- `student/pseudo_data_sk_daynight.csv`: day-night pseudo-data.
- `student/solar_neutrino_student_data.zip`: all student CSV files and dataset notes in one archive.
- `teacher/peanuts_solar_energy_spectrum.csv`: PEANUTS continuum spectra multiplied by source flux.
- `teacher/peanuts_solar_line_sources.csv`: PEANUTS line-source markers for the lecture spectrum plot.
